# README.md for views/tenants/mobilemeadows

## Purpose
This directory contains the implementation details for integrating Mobile Meadows into Slab's tenant management system. The primary goals are to update bookings on the admin panel to reflect the Mobile Meadows calendar and booking system, and to ensure a custom calculator element is available.

## Key Files
- `booking-admin-update.js`: Updates admin bookings to mimic Mobile Meadows.
- `calculator-custom-element.js`: Custom implementation for the calculator.
- `mobile-meadows-template.js`: Template for provisioning Mobile Meadows as a tenant in Slab.

## Quick Reference
1. **Update Admin Bookings**:
   - Use `booking-admin-update.js` to sync with Mobile Meadows' calendar and booking system.
2. **Custom Calculator Element**:
   - Implement the custom calculator logic using `calculator-custom-element.js`.
3. **Provision Tenant**:
   - Adopt a template from `mobile-meadows-template.js` for provisioning as a tenant in Slab.

For further details, refer to the respective file documentation or project issue tracking system.
