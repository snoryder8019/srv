# Admin Panel

Access your admin panel at `/admin`. Sign in with a Google account that has admin access.

## Dashboard

The main dashboard shows:
- Quick stats (portfolio items, clients, open invoices, blog posts, pages)
- The Master AI Agent chat — ask it to write blog posts, update copy, or create pages

## Sidebar Navigation

| Section | What It Does |
|---------|-------------|
| **Dashboard** | Stats overview + AI assistant |
| **Portfolio** | Manage portfolio items and images |
| **Blog** | Write and publish blog posts |
| **Pages** | Create custom pages (content, landing, data-list) |
| **Sections** | Add custom sections to your landing page |
| **Site Copy** | Edit all text on your landing page |
| **Design & Settings** | Colors, fonts, logos, visibility toggles |
| **Assets** | Upload and organize files and media |
| **Clients** | Manage client records |
| **Meetings** | Create and manage video meetings |
| **Bookkeeping** | Invoices, payments, ledger |
| **Email Marketing** | Campaigns, contacts, tracking |
| **Users & Permissions** | Manage team member access |
| **Settings & Keys** | Business profile, API keys, integrations |

## AI Agent

Every content editor (Blog, Copy, Sections, Pages) has a slide-in AI panel. Click the AI button to:
- Ask for content suggestions
- Generate full drafts
- Research topics with live web search
- Auto-fill form fields (highlighted in green, revertible)

The Master Agent on the Dashboard can route tasks to any module — just describe what you want.

## Profile

Click your name in the sidebar footer to access profile settings, tutorial preferences, and account options.
