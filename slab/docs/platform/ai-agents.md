# AI Agents

Every content editor in your admin panel has a built-in AI assistant powered by web search and language models.

## How to Use

1. Open any content editor (Blog, Copy, Sections, Pages)
2. Click the **AI** button to open the agent panel
3. Type what you want — e.g. "Write a blog post about spring marketing trends"
4. The agent researches your topic online, then generates content
5. Review the output — fields auto-fill with a green highlight
6. Click the revert button on any field to undo the AI suggestion

## Master Agent (Dashboard)

The Dashboard has a Master Agent that can route tasks to any module:
- "Write a blog post about..." → Blog Agent
- "Update the hero section to..." → Copy Agent
- "Create a FAQ section about..." → Section Agent
- "Build a landing page for..." → Page Agent

Results can be:
- **Applied directly** to your site
- **Opened in the editor** for review before saving
- **Followed up** with contextual suggestion chips

## What the AI Knows About You

The AI reads your **Business Profile** from Settings to tailor content:
- Business name, type, industry
- Services offered
- Location and service area
- Target audience
- Brand voice (e.g. "professional and friendly")

Keep your Business Profile up to date for the best AI output.

## Web Search

Agents use Brave Search to find current information before writing. This means:
- Blog posts include real, current data
- Content references actual trends and statistics
- The AI doesn't rely solely on training data
