# Site Copy

Edit all landing page text at `/admin/copy`.

## Sections
- **Hero** — Eyebrow, heading, emphasis text, subheading, badge
- **Services** — Label, heading, subheading, 3 service titles + descriptions
- **About** — Quote, description, signature
- **Process** — Label, heading, 4 step titles + descriptions
- **Contact** — Subheading, location, serving area

## AI Agent
Click the AI button and describe what you want — e.g. "Make the hero more energetic" or "Rewrite services for a dental practice." The agent fills multiple fields at once.

## How It Works
Copy is stored as key-value pairs. The public landing page reads these values and renders them in the appropriate sections. Default placeholder text shows until you customize it.
