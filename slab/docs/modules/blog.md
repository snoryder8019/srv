# Blog

Create and publish blog posts at `/admin/blog`.

## Creating a Post
- **Title** — The headline
- **Slug** — URL path (auto-generated from title, editable)
- **Excerpt** — Short summary for listings
- **Content** — Full HTML editor
- **Featured Image** — Upload or paste URL
- **Category & Tags** — For organization and filtering
- **Status** — Draft or Published

## AI Agent
Click the AI button to generate a full blog post. The agent:
1. Searches the web for current information on your topic
2. Writes title, excerpt, HTML content, category, and tags
3. Auto-fills the form fields (green highlight, revertible)

## Public Display
- Blog listing at `/blog`
- Individual posts at `/blog/{slug}`
- Latest 3 posts shown on home page (if Blog section is visible in Design)
- Included in `/sitemap.xml`
