# Design & Theming

Customize your site's look at `/admin/design`.

## Color Tokens
| Token | Purpose |
|-------|---------|
| Primary | Main brand color (nav, headings, buttons) |
| Primary Deep | Darker shade (nav background) |
| Primary Mid | Medium shade (hover states) |
| Accent | Highlight color (CTAs, links, emphasis) |
| Accent Light | Lighter accent (hover, subtle highlights) |
| Background | Page background |

## Font Tokens
- **Heading Font** — Used for all headings (Google Fonts)
- **Body Font** — Used for paragraphs and UI text (Google Fonts)

## Visibility Toggles
Show or hide landing page sections: Hero, Services, Portfolio, About, Process, Reviews, Contact, Blog.

## Agent Settings
- **Agent Name** — The name shown in AI chat panels
- **Agent Greeting** — The opening message when chat opens

## Layout Options
- **Portfolio Layout** — Grid or list
- **Blog Layout** — Grid or list
- **Nav Logo Display** — Text, image, or both

## Themes
Save your current design as a theme and switch between saved themes. Useful for seasonal variations or A/B testing.

## Brand Images
Upload logos at different sizes:
- **Primary Logo** — Main logo
- **White Logo** — For dark backgrounds (footer)
- **Icon** — Favicon or small format
