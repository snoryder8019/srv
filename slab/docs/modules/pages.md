# Pages

Create custom pages at `/admin/pages`. Pages live at `/{slug}` on your public site.

## Page Types

### Content Page
Free-form HTML editor. Good for policies, about pages, info pages.

### Landing Page
Visual block builder with drag-and-drop reordering. Block types:

| Block | Fields |
|-------|--------|
| Hero | Heading, subheading, CTA button |
| Text | Heading, subheading, body (HTML) |
| Split | Text + image side-by-side with CTA |
| CTA | Bold call-to-action banner |
| Cards | Heading + up to 4 feature cards |
| FAQ | Heading + up to 5 expandable Q&A pairs |

### Data List
Auto-paginated card grid pulling from Blog or Portfolio collections.

## SEO Fields
Each page has: meta title, meta description, robots directive, sitemap priority, change frequency, canonical URL, and Open Graph image.

## AI Agent
The Page Agent generates content based on page type — HTML body for content pages, suggested block layout for landing pages, title + SEO for data-list pages.

## Navigation
Toggle **Show in Nav** to add the page to the site navigation bar.
