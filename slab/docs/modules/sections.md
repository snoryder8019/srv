# Sections

Add custom sections to your landing page at `/admin/sections`. Sections render between the Contact section and Footer.

## Templates

| Template | Layout |
|----------|--------|
| Text | Full-width heading + body copy |
| Split | Text left, image right with optional CTA |
| CTA | Bold call-to-action banner with optional background image |
| Cards | Heading + up to 4 feature cards with optional images |
| FAQ | Expandable Q&A accordion |

## Managing Sections
- Toggle visibility per section
- Reorder by creation date
- Edit content and images inline

## AI Agent
The Section Agent generates content for any template type. Describe what you want and it fills the fields for your chosen template.
