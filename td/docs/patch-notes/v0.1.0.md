# v0.1.0 - Initial Boilerplate

**Date:** May 1, 2026
**Status:** Foundation laid

## Added

- Express + EJS + Socket.IO + Mongoose stack on port 3700
- Domain wired to `towers.madladslab.com`
- Hex grid math (axial coords, pointy-top) - server + client
- Tower / Map / Run Mongoose models with voting and moderation fields
- REST API at `/api/v1/` with health, towers, maps, runs, uploads
- GLTF upload endpoint with size + extension validation
- Three.js scene boilerplate (`three/scene.js`)
- Hex board renderer (`three/hex-board.js`) with role-based coloring
- Tower builder UI (upload GLTF → preview → save draft)
- Map builder UI (paint spawn/base/path/blocked → save draft)
- Play view (renders demo map)
- Browse community page (lists approved towers and maps)
- Terminal-aesthetic stylesheet

## Architecture Decisions

- **Compartmentalization first**: every concern has one folder. See `docs/architecture/OVERVIEW.md`.
- **Hex math duplicated server + client** - both must agree, both get tested.
- **Schema-first**: every entity has a Mongoose model before it has a UI.
- **Low-code today = forms**: Blockly visual scripting deferred to v0.3.x.
